# <img src="img/imdb-icon-white-corn.png" alt="IMDb Suggest Alfred Workflow Icon" width="45" align="center"/> IMDb Suggest | Alfred Workflow

Get instant search suggestions from IMDb as you type in Alfred. Fast, real-time autocomplete for movies, TV shows, actors, and more with poster previews and Quick Look support.

This workflow complements Alfred's built-in IMDb Web Search by providing instant autocomplete suggestions with visual previews, making it faster to find and explore IMDb content.

## Download

- Available on the Alfred Gallery. [Get it here](https://alfred.app/workflows/vanstrouble/imdb-suggest/).
- You can download it directly [from GitHub here](https://github.com/vanstrouble/imdb-suggest-alfred-workflow/releases).

## Usage

Start typing your search query in Alfred using your configured keyword (default: `imdb` or your preferred trigger).

- **Keyword:** `[your-keyword] [search query]`
- **Action:** Press `Enter` to open IMDb page in your browser
- **Quick Look:** Press `⇧` (Shift) or `⌘Y` to preview the IMDb page
- **Copy ID:** Hold `⌘` (Command) and press `Enter` to copy the IMDb ID

#### Examples:

| Query Input          | Result                                          |
|----------------------|-------------------------------------------------|
| `inception`          | Shows: Inception (2010), Inception: The Cobol Job, etc. |
| `breaking bad`       | Shows: Breaking Bad (TV Series), actors, related content |
| `tom hanks`          | Shows: Tom Hanks, his movies, TV appearances, etc. |
| `the godfather`      | Shows: The Godfather trilogy, cast, related content |
